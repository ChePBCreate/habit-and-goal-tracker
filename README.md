# Habit and Goal Tracker

# Description:
A simple habit and goal tracking calendar created using Javascript, HTML, and CSS. Helps to visualise and track progress on personal goals or habits.

## Features
- Allows users to set and edit a goal or habit title.
- Clickable days to mark down goal or habit progression.
- Progress is saved using browser localStorage.
- Automatically displays and updates the current date.
- Reset button to clear all saved progress.
- Pop up message of encouragement once several days in a row are completed.
- Hand drawn background.

## How to Run
1. Clone or download this repository.
2. Open the project folder.
3. Open the 'habittracker.html' file in a web browser.
4. The calendar will automatically load.

## Requirements
- Modern web browser.
- Javascript enabled.
- No external libraries or frameworks required.
